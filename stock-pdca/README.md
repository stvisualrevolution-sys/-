# 朝株ラボ

毎朝8:00に、前回予想の検証 + 当日予想 + 改善点をまとめて出すための運用ディレクトリ。

愛称: 朝株ラボ
正式には、日経平均の寄り前分析を毎日改善しながら回すPDCA運用です。

使い方の呼び名例:
- 「朝株ラボ、今日どう？」
- 「朝株ラボの週次レビュー見せて」
- 「朝株ラボの月次レビューお願い」

## 目的
- 日経平均の当日終値ベース騰落を毎朝寄り前に予想する
- 翌朝、その予想の方向・変動幅・根拠の当たり外れを検証する
- 小さく改善点を反映し、PDCAを回す

## 判定ルール
- 予想対象: 前営業日終値 → 当営業日終値 の日経平均騰落率
- 上昇: +0.3%以上
- 下落: -0.3%以下
- 横ばい: -0.3%超 〜 +0.3%未満

## 保存先
- `data/` 予想・結果・改善ログ
- `reports/` 日次レポート
- `scripts/` 実行スクリプト

## 実行
- 毎朝8:00のcronから `scripts/run_daily_report.sh` を実行
- 必要に応じて手動実行も可能

## 注意
これは投資助言ではなく、観測・仮説・検証・改善のための分析運用です。
精度改善を目指しますが、利益や的中を保証するものではありません。
